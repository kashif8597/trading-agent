# Trading Agent

A Python-based trading agent prototype built to explore agentic artificial intelligence for market monitoring and decision support.

## Current Features

- Watchlist tracking
- Simple market signal detection
- Moving average comparison logic
- Console-based interaction
- Git and GitHub version control

## Current Watchlist

- VOO
- SPY
- NVDA

## How It Works

The script checks sample or live market data and classifies symbols into simple signal states:

- GREEN
- YELLOW
- RED

This is an early prototype focused on learning, structure, and future expansion into a more complete trading assistant.
